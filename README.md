FPGA-Based Fetal Heartbeat Detection System
📌 Overview

This project focuses on real-time fetal heartbeat detection using Phonocardiogram (PCG) signals. It combines signal denoising, analysis, and hardware implementation on FPGA to achieve fast and accurate biomedical monitoring.

🎯 Objectives
Extract fetal heart signals from noisy PCG data
Apply signal processing techniques for denoising
Implement real-time detection on FPGA hardware
Improve accuracy and speed compared to software-only systems
🛠️ Tools & Technologies
MATLAB – Signal preprocessing, filtering, and analysis
Intel Quartus Prime – FPGA design and implementation
FPGA Board – Hardware acceleration for real-time processing
⚙️ Methodology
Signal Acquisition
Collect PCG signals containing maternal and fetal heart sounds
Signal Denoising
Apply filtering techniques to remove noise and interference
Feature Extraction
Identify key characteristics of fetal heart signals
FPGA Implementation
Convert algorithms into hardware logic using Quartus
Real-Time Detection
Detect fetal heartbeat efficiently with low latency
📊 Key Features
Real-time processing using FPGA
Improved detection accuracy
Noise reduction in biomedical signals
Hardware-software co-design approach
📈 Applications
Prenatal health monitoring
Biomedical signal processing systems
Portable fetal monitoring devices
🚀 Future Enhancements
Integration with IoT for remote monitoring
Machine learning-based signal classification
Mobile app for real-time tracking
